Behind-the-scenes

System consists of twenty Itek 245W solar photovoltaic modules feeding a grid-tied Eltek THEIA 4.4 UL inverter. This equipment is Washington manufactured, qualifying us for some lucrative state subsidies in addition to the monthly power bill savings.

I built this custom display/API to help me share this data online. The live display uses Server-Sent Events from node.js; the charts are from data logged into CouchDB, loaded via my Fermata library, and displayed using D3.js to generate SVG graphics.
